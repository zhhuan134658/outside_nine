# demo

vuecli-4 基础项目

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve //本地测试环境
```

### Compiles and minifies for production

```
npm run build //测试服务器环境
npm run build:prod //线上服务器环境
npm run build-report //生成分析图
```

### Lints and fixes files

```
npm run lint
```
